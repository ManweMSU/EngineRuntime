 Таблица строк представляет из себя словарь пар идентификатор-строка. Идентификатор является числом. Данный формат имеет своей целью хранение строк на диске, упрощение локализации программ.

Для хранения данных на диске рекомендуется использовать расширение файла .ecs или .ecst.

Заголовок. Расположен по смещению 0 в файле.
